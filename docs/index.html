An overview¶

cryptocompsdk¶

This is a simple example:

from cryptocompsdk import CryptoCompare
API_KEY = 'my-api-key'
cc = CryptoCompare(API_KEY)

data = cc.history.get(from_symbol='BTC', to_symbol='USD', exchange='Kraken')
df = data.to_df()

cryptocompsdk.main.CryptoCompare(api_key[, …])
